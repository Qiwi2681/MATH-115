# 6.4 Logarithmic functions

1. Log function with base $a$ 

    $y=log(a^x) == a^y$

    "$x$ is the exponent on $a$ that gives $y$"

## change to log form:

- a) $2^y = 8$

    $y = log{_2}8$

- b) e^3 = y

    $e = log{_e}y$

- c) $\pi^x = 8$

    $x = log{_\pi}8$

- d) $x^a = y$

    $a = log{_x}y$

## change to exponential form

- a) $w^3 = log{_3}27$

    $w^3 = 27$

- b) $x = log{_e}4$

    $e^x = 4$

- c) $2 = log{_4}16$

    $4^2 = 16$

- d) $y = log{_2}16$

    $2^y = 16$
